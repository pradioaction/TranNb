   ## 控件间信号通信关系

   | 发送方         | 信号                  | 接收方       | 作用                     |
   | -------------- | --------------------- | ------------ | ------------------------ |
   | BaseCell       | `selected`            | CellManager  | 选中单元格               |
   | BaseCell       | `translate_requested` | CellManager  | 请求翻译                 |
   | BaseCell       | `delete_requested`    | CellManager  | 请求删除                 |
   | BaseCell       | `move_up_requested`   | CellManager  | 请求上移                 |
   | BaseCell       | `move_down_requested` | CellManager  | 请求下移                 |
   | MarkdownEditor | `content_changed`     | CellManager  | 内容变化通知             |
   | MarkdownEditor | `needs_height_update` | MarkdownCell | 需要更新高度             |
   | CellManager    | `content_changed`     | MainWindow   | 内容变化（标记文件修改） |

   ## 核心工作流程

   1. **用户选择单元格**：点击单元格 → `BaseCell.selected` → `CellManager.on_cell_selected()`
   2. **用户点击翻译**：点击翻译按钮 → `BaseCell.translate_requested` → `CellManager.on_cell_translate_requested()` → `MarkdownCell.translate()`
   3. **内容编辑**：在 MarkdownEditor 编辑 → `MarkdownEditor.content_changed` → `CellManager.content_changed` → MainWindow 标记文件已修改
   4. **高度调整**：内容变化 → `MarkdownEditor.needs_height_update` → `MarkdownCell._on_needs_height_update()` → `adjust_height()`

   这种设计实现了良好的**关注点分离**：
   - `BaseCell` 负责通用的 UI 和操作（侧边栏、选择状态）
   - `MarkdownCell` 负责特定的 Markdown 编辑功能
   - `MarkdownEditor` 是可复用的 Markdown 编辑组件
   - `CellManager` 负责整体管理和协调
